Dark Platformer Game Structure

This is a simple, atmospheric platformer game inspired by Limbo and Inside. The game features a stick figure character navigating through a dark, shadowy environment.

Note: To run this game, you need to place a background image file named background.png in the assets/images/ directory.

Directory Structure

platformer-game/
├── index.html         # Main HTML file
├── js/
│   └── main.js        # Game logic
└── assets/
    └── images/
        └── background.png  # Background image (needs to be added)

Game Features

  • Simple stick figure character
  • Side-scrolling platformer mechanics
  • Jumping and double jumping
  • Parallax scrolling background
  • Atmospheric dark environment

Controls

  • A/Left Arrow: Move left
  • D/Right Arrow: Move right
  • W/Up Arrow/Space: Jump (press again while in air for double jump)
  • Escape: Return to main game

Integration with Main Game

This platformer game is accessed through the pentagram portal in the main game. When the player touches the pentagram in the bottom left corner of the map, they are transported to this platformer game.

When the player reaches the end of the platformer level or presses Escape, they are returned to the main game.
